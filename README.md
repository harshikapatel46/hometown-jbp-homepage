# hometown-jbp-homepage
A responsive tourism homepage showcasing the beauty,  and street food of Jabalpur using HTML and CSS.

This project is a simple and aesthetic tourism website based on my hometown Jabalpur.

## ✨ Features
- Clean and responsive design
- Image hover effects
- Sections for attractions, street food, and nature

## 🛠️ Tech Used
- HTML
- CSS

## 🎯 Purpose
This project was created to practice web development and design a visually appealing homepage.
